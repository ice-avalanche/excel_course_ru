# Урок 16. Применение нейросетей в проектах

## 1. Назначение

Современные инструменты искусственного интеллекта позволяют существенно ускорить работу в Excel:

- автоматизировать анализ данных;
- генерировать формулы и код VBA;
- строить прогнозы;
- анализировать текстовые данные;
- готовить отчёты и выводы.

Цель занятия – показать, как использовать ИИ как инструмент аналитика, а не как «чёрный ящик».

## 2. Роль ИИ в аналитике

ИИ может:

- объяснить формулу;
- предложить корректировку расчёта;
- выявить аномалии;
- помочь написать макрос;
- сгенерировать структуру отчёта;
- интерпретировать результаты регрессии.

## 3. Генерация формул

Пример задачи:  
«Найти средний чек по клиенту без учёта возвратов».

ИИ может предложить формулу:

```excel
=СРЗНАЧЕСЛИМН(B:B;C:C;"<>Возврат")
````

Пример более сложной формулы:

```excel
=СУММЕСЛИМН(D:D;A:A;F2;B:B;">="&H2;B:B;"<="&I2)
```

Использование ИИ:

* формулировать задачу максимально конкретно;
* проверять диапазоны;
* тестировать на малых данных.

## 4. Генерация VBA-кода

Пример запроса:
«Создай макрос, который обновляет сводные таблицы и сохраняет файл в PDF».

ИИ может предложить:

```vba
Sub UpdateAndSave()

    ThisWorkbook.RefreshAll
    
    ActiveWorkbook.ExportAsFixedFormat _
    Type:=xlTypePDF, _
    Filename:="C:\Report.pdf"
    
    MsgBox "Отчет обновлен и сохранен."

End Sub
```

Важно:

* проверять код перед использованием;
* тестировать на копии файла.

## 5. Анализ текста

ИИ может:

* классифицировать комментарии клиентов;
* выделять ключевые слова;
* группировать обращения по темам;
* анализировать тональность.

Пример задачи:

* Есть столбец «Комментарий».
* Нужно определить: положительный / нейтральный / отрицательный.

Алгоритм:

1. Экспортировать данные.
2. Использовать ИИ для классификации.
3. Импортировать результат обратно в Excel.

## 6. Прогнозирование

ИИ может:

* построить модель прогноза;
* предложить признаки;
* оценить сезонность;
* сравнить несколько моделей.

Пример:

* Данные продаж по месяцам.
* Запрос: «Построй прогноз на 6 месяцев».

ИИ может:

* предложить линейную регрессию;
* объяснить MAPE;
* сравнить тренд и сезонность.

## 7. Проверка гипотез и интерпретация моделей

Пример:

Excel выдал:

* R² = 0,82
* p-value = 0,03

ИИ может помочь интерпретировать:

* модель объясняет 82% вариации;
* коэффициент статистически значим;
* проверить мультиколлинеарность.

При этом важно помнить, что ИИ не заменяет понимание статистики.

## 8. Ограничения ИИ

| Ограничение          | Риск                       |
| -------------------- | -------------------------- |
| Галлюцинации         | Некорректные формулы       |
| Ошибки в логике      | Неверные расчёты           |
| Отсутствие контекста | Неправильная интерпретация |
| Конфиденциальность   | Утечка данных              |

## 9. Правила безопасного использования

* Не передавать конфиденциальные данные.
* Проверять формулы вручную.
* Тестировать код на копии.
* Использовать ИИ как помощника, а не как замену анализа.
